# 如何参与

本教程使用lyx 2+编写，搭建书写环境的步骤是：
 * 建议安装Linux环境
 * 安装textlive最新版和lyx 2+
 * git clone https://github.com/subaochen/java-tutorial
 * 执行common中的deploy.sh脚本安装lyx的几个模块，然后启动lyx，点击“工具”->"重配置"，重启lyx
 * 执行./makebook.sh可以生成本书的pdf文件，请到上一级目录查看生成的pdf文件。

  happy writting now:-)
